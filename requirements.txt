## **5. Render'da Deploy**

1. Dashboard → **New +** → **Web Service**
2. **Connect a repository** → `konum-server` seç → **Connect**
3. Şu ayarları yap:

| Alan | Değer |
|------|-------|
| **Name** | `konum-server` |
| **Region** | Frankfurt (EU) |
| **Branch** | main |
| **Runtime** | Python 3 |
| **Build Command** | `pip install -r requirements.txt` |
| **Start Command** | `uvicorn server:app --host 0.0.0.0 --port $PORT` |
| **Instance Type** | **Free** |

4. **Create Web Service** butonuna bas
5. 2-3 dakika bekle...
